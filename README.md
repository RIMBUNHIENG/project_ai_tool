## Run Locally
**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Set the `Groq_api_key` in [.env.local](.env.local) to your Gemini API key
4. Set the `Hugging face` in [.env.local](.env.local) to your Hugging face

5. Run the app:
   `npm run dev`
